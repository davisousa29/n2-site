O que gostaria de acessar?

Este site foi criado pelos alunos Adriano Lucas Dutra Caixeta, Davi Ramos de Sousa e Fellipe Dias Lourenço de Oliveira, como parte do projeto final da disciplina de Complexidade de Algoritmos.

O objetivo deste site é fornecer aos alunos da UNIALFA uma introdução à disciplina de complexidade de algoritmos. A complexidade de um algoritmo é uma medida do tempo que o algoritmo leva para resolver um problema, em função do tamanho da entrada.

A complexidade de algoritmos é uma área importante da ciência da computação, pois determina a solvabilidade prática de problemas computacionais. Problemas com complexidade alta são difíceis de resolver, mesmo para computadores modernos. Problemas com complexidade baixa são mais fáceis de resolver e podem ser resolvidos em tempo razoável, mesmo para entradas de tamanho grande.

O site foi desenvolvido utilizando as tecnologias HTML, CSS e JavaScript. Os alunos utilizaram o framework Bootstrap para facilitar o desenvolvimento do layout do site.
