Paradigmas de Programação

Este trabalho apresenta um site sobre paradigmas de programação, desenvolvido pelos alunos Adriano Lucas, Davi Ramos e Fellipe Dias, para a disciplina de Paradigmas de Programação do Centro Universitário Alves Faria - UNIALFA. O site foi desenvolvido utilizando as tecnologias JavaScript, HTML e CSS.

O site sobre paradigmas de programação é uma ferramenta útil para estudantes e profissionais de desenvolvimento de software. O site fornece uma visão geral dos principais paradigmas de programação e pode ser usado para aprender sobre cada paradigma de forma detalhada.
