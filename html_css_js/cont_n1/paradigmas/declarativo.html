Paradigma Declarativo

O paradigma lógico é um paradigma de programação que utiliza a lógica matemática para expressar programas. Em vez de descrever como um programa deve ser executado, o paradigma lógico descreve o que o programa deve fazer.

As linguagens de programação lógica são baseadas na lógica de predicados, que é uma forma de lógica matemática que permite expressar declarações sobre objetos e seus relacionamentos. Um programa lógico é uma coleção de declarações de fatos e regras. As declarações de fatos descrevem o mundo do programa, enquanto as regras descrevem como o mundo do programa pode mudar.

Algumas linguagens de programação lógica populares incluem:

→ Datalog

→ Curry
