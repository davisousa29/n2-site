Paradigma Funcional

O paradigma funcional é um paradigma de programação que trata a computação como uma avaliação de funções matemáticas e que evita estados ou dados mutáveis. Ela enfatiza a aplicação de funções, em contraste da programação imperativa, que enfatiza mudanças no estado do programa.

As linguagens de programação funcionais são baseadas no conceito de função matemática, em que para cada elemento do seu conjunto domínio (entrada) há apenas um elemento no seu conjunto contradomínio (saída). Programas escritos nele são definições de funções e de especificações de aplicação destas, e as execuções consistem em avaliá-las.

Algumas linguagens de programação funcionais populares incluem:

→ Haskell

→ Scheme

→ Lisp
