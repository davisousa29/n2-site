Scheme

Scheme é uma linguagem de programação de alto nível, interpretada, de tipagem dinâmica e multiparadigma. Foi criada por Guy L. Steele e Gerald Jay Sussman no outono de 1975, a partir da linguagem Lisp com o intuito de estudar a teoria dos atores de Carl Hewitt.

Este código define uma função chamada factorial que recebe um número como entrada e retorna o fatorial desse número. O fatorial de um número é o produto de todos os números inteiros de 1 a esse número.

A saída deste comando será: 120.

Este código é um exemplo simples de como Scheme pode ser usado para definir funções. Scheme também pode ser usado para definir procedimentos, estruturas de dados, e muito mais.
