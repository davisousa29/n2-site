Algoritmos Gulosos

Algoritmos gulosos são usados em problemas de otimização onde é interessante realizar um conjunto de melhores soluções locais, possuindo como objetivo obter uma solução ótima global, tomando como base um determinado parâmetro. Os parâmetros que definem o que será a melhor solução durante a iteração irão variar para cada problema.

Eles são uma técnica de projeto de algoritmos que tenta resolver um problema fazendo a escolha localmente ótima em cada fase, com a esperança de encontrar uma solução global ótima.

A ideia básica de um algoritmo guloso é tomar decisões com base apenas na informação disponível no momento, sem se preocupar com as consequências futuras dessas decisões. Em outras palavras, um algoritmo guloso nunca reconsidera as decisões tomadas, independentemente das consequências. Podemos então citar algumas das suas principais características:

-As escolhas realizadas são definitivas;

-Não levam em consideração as consequências de suas escolhas;

-Podem fazer cálculos repetitivos.

Uma das suas principais vantagens é que são algoritmos simples e de fácil implementação. Uma das principais desvantagens é que eles podem entrar em loop, consequentemente desenvolvendo um caminho infinito.

Aqui estão alguns exemplos de algoritmos gulosos:

- Algoritmo de Dijkstra: encontra o caminho mais curto entre dois pontos em um grafo.

- Algoritmo de Prim: encontra a árvore geradora mínima de um grafo.

- Algoritmo de Ford-Fulkerson: encontra o fluxo máximo em um canal.
