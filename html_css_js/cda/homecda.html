Complexidade de Algoritmos

A complexidade de algoritmos é uma área importante da ciência da computação, pois determina a solvibilidade prática de problemas computacionais. Problemas com complexidade alta são difíceis de resolver, mesmo para computadores modernos. Problemas com complexidade baixa são mais fáceis de resolver e podem ser resolvidos em tempo razoável, mesmo para entradas de tamanho grande.
