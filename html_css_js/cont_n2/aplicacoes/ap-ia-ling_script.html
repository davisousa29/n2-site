Aplicações de IA

As aplicações de IA são aquelas que usam a inteligência artificial para realizar tarefas. Elas podem ser usadas para automatizar tarefas, tomar decisões, gerar conteúdo e interagir com os usuários. São linguagens de programação que utilizam de computação simbólica ao invés de numérica, ou seja, ela utiliza de uma longa lista de conhecimentos e não de dados para chegar no resultado solicitado no início.

Linguagens que foram utilizadas para aplicações de IA:

LISP

Prolog


Linguagens de Script

As linguagens de script são linguagens de programação de alto nível que são usadas para automatizar tarefas e gerar conteúdo. Elas são frequentemente usadas para desenvolvimento web, desenvolvimento de aplicativos móveis e automação de tarefas. São listas de comandos que são executados sequencialmente para realizar uma ação. Como exemplo um arquivo BATCH com uma lista de comandos inserida no mesmo, utilizado para apagar arquivos da pasta temp do computador para liberar armazenamento em um clique.

Linguagens que foram utilizadas para linguagens de script:

Python

Action Script
