Aplicações Científicas

Aplicações que são usadas para realizar cálculos científicos e técnicos, como simulações, análise de dados com um grande número de operações aritméticas com ponto-flutuante e engenharia.

Linguagens que foram utilizadas para aplicações científicas:

Algol

Fortran


Aplicações Comerciais

Aplicações que são usadas para fins comerciais como automatizar tarefas e processos de negócios, como sistemas de ERP, CRM e e-commerce. Essas linguagens possuem facilidade para produzir relatórios e armazenar dados.

Linguagens que foram utilizadas para aplicações científicas:

COBOL

Clipper
